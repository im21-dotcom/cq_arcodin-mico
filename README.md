Este código se trata de um algoritmo em Python associado à biblioteca Streamlit que foi desenvolvido para auxiliar os cálculos de dose em controles de qualidade de paciente específico.

Foi pensando para o controle de qualidade feito com água sólida e uma câmara de ionização em tratamentos com arco dinâmico feitos no Hospital de Clínicas de Porto Alegre.

A interface de utilização do algoritmo pelo usuário está disponível em: https://cq-arcodinamico-tvnnfhutr8ushhwevcbk2c.streamlit.app/

Os valores de KQ,Q0 utilizados para os feixes de 6 MV e 10 MV algoritmo estão de acordo com o TRS-398 (IAEA) para a câmara de ionização IBA FC-65P.
